# ED_Lista02.03
## Busca Recursiva num Vetor

Este desafio consiste em criar uma aplicação, em Java, que identifique a quantidade de números pares presentes num vetor.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A busca deve ser feita por uma função recursiva;
3. O vetor e seu tamanho devem ser enviados como parâmentros da função;
4. Declarar, via comentário, a condição de parada da função.
```
